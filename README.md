Sephora E-Commerce Clone
A front-end web development project that replicates the core UI/UX of the Sephora homepage. This project focuses on responsive design, interactive product carousels, and a clean, modern aesthetic using Bootstrap 5.

 Key Features
Announcement Bar: A top bar for promotional alerts with custom background colors.

Dynamic Navigation: A dual-level navigation system including a search bar, user account links, and a category-based secondary menu.

Full-Width Hero Carousel: An interactive banner section with multi-item slides, featuring customized background colors for different product categories (e.g., Pink, Yellow, Cyan).

Product Carousel: A second carousel designed specifically for product cards, maintaining fixed dimensions while supporting full-width display.

Responsive Grid: Product listings are organized using the Bootstrap grid system to ensure compatibility across different screen sizes.

 Technologies Used
HTML5: Semantic structure of the web pages.

CSS3: Custom styling, layout fixes for full-width components, and UI enhancements.

Bootstrap 5: Utilized for the grid system, navigation bars, carousels, and utility classes (container-fluid, bg-warning, etc.).

Font Awesome: Integrated for high-quality icons (Search, Shop, User, Basket).

Google Fonts / Custom Typography: To match the high-end brand identity.
